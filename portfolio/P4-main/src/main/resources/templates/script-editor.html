Wait Command

Pause execution for a specified duration

wait 5 sec;
wait 2 minutes;

Count-Based Repeat

Loop commands a specific number of times

repeat 3 times { lights on; wait 2 sec; lights off; wait 1 sec; }
repeat 5 times { brightness 100; wait 1 sec; brightness 50; wait 1 sec; }
Time-Based Repeat

Loop commands for a specific duration

repeat 30 seconds { lights color "red"; wait 1 sec; lights color "blue"; wait 1 sec; }
repeat 5 minutes { brightness 100; wait 3 sec; brightness 20; wait 3 sec; }
Nested Loops

Time-based outer with count-based inner loops

repeat 2 minutes { // Color sequence lights color "red"; wait 1 sec; // Inner count-based loop repeat 3 times { brightness 100; wait 200 ms; brightness 20; wait 200 ms; } lights color "blue"; wait 1 sec; }

Scene Definition

Define and use reusable scenes

Define Scene

Create reusable lighting scenes

define scene movieMode { brightness 30; lights color "warm"; }
define scene partyLights { brightness 100; repeat 3 times { lights color "red"; wait 1 sec; lights color "blue"; wait 1 sec; } }
Invoke Scene

Activate a defined scene

scene movieMode;
scene partyLights;
Define Light Groups

Create and control groups of lights

define group livingRoom [1, 2, 3];
group livingRoom on;
group livingRoom brightness 75;
group livingRoom color "blue";
group livingRoom transition "red" to "green" over 5 seconds;
